# fetzerch.github.io

This is my blog.

Design based on [Poole](http://getpoole.com/) by
[@mdo](https://twitter.com/mdo).

## Run locally

Build and run the site locally at <http://localhost:4000/>:

```text
rake serve
```

## Validate generated HTML output

Validate generated HTML output with html-proofer:

```text
rake test
```
